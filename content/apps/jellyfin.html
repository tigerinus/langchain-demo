Intro

Jellyfin enables you to collect, manage, and stream your media. Run the Jellyfin server on your system and gain access to the leading free-software entertainment system, bells and whistles included.

 

Install

  • CasaOS App Store
  • CasaOS Custom Install App
    • AppFile: Jellyfin.json

 

Usage

https://jellyfin.org/docs/general/quick-start.html

 

Basic Info

  • Docker Image: linuxserver/jellyfin
  • Supported Architectures:
    • x86-64
    • arm64
    • armhf
  • Dependencies: Nothing

 

Settings

Ports

  • HTTP WebUI
    • Container: 8096 TCP
  • HTTPS WebUI (Optional)
    • Container: 8920 TCP
  • DLNA LAN Auto-discovery (Optional)
    • Container: 1900 UDP
    • Host: 1900 UDP
  • Jellyfin LAN Auto-discovery (Optional)
    • Container: 7359 UDP
    • Host: 7359 UDP

 

Volumes

  • Jellyfin Data
    • Jellyfin data storage location. This can grow very large, 50gb+ is likely for a large collection.
    • Container: /config
  • Media Data
    • Media goes here. Add as many as needed e.g. /data/movies, /data/tvshows, etc.
    • Container: /data/*

 

Environment variables

  • TimeZone (Optional)
    • Specify a timezone to use (e.g. Europe/London)
    • Key: TZ
    • Value: Europe/London
  • UserID (Optional)
    • Key: PUID
    • Value: 1000
  • GroupID (Optional)
    • Key: PGID
    • Value: 1000
